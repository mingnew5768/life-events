如何更新網站（拿到新的 index.html 之後）
----------------------------------------

1. 把新的 index.html 複製到這個資料夾，
   直接覆蓋舊檔案即可。

2. 開啟 PowerShell，貼上以下指令，切換到這個資料夾：

   cd D:\工具開發\腳本工具\life-events

3. 依序執行這三行，把更新推上 GitHub：

   git add index.html
   git commit -m "update"
   git push

4. 如果跳出瀏覽器要求授權登入，
   點「Authorize git-ecosystem」完成授權即可。

5. 等待 1-2 分鐘後，重新整理網站頁面即可看到更新
   （建議按 Ctrl + Shift + R 強制重新整理，避免看到舊版本）

如何測試（推上線之前，建議先做）
----------------------------------------

直接在這個資料夾裡雙擊 index.html，
會用瀏覽器打開，可直接測試功能是否正常。

測試項目建議：

- 編輯 / 刪除事件
- 列印預覽（A4 橫向，確認備註是否正確跟隨/換頁）
- 備份 / 匯入 JSON
- 辨識匯入（拖圖片、PDF，或貼文字）


----------------------------------------
辨識匯入功能：需要 API Key
----------------------------------------

「辨識匯入」（不管是上傳照片/PDF辨識，還是直接貼文字整理），
背後都是呼叫 Anthropic 的 AI 模型，需要自己的 API Key 才能使用。

【怎麼拿 API Key】
1. 前往 https://console.anthropic.com 註冊/登入
   （這跟 Claude.ai 帳號是不同系統，要另外申請）
2. 左側選單「API Keys」→「Create Key」
3. 複製產生的 sk-ant-... 字串（只顯示一次，要馬上存起來）
4. 注意：這是用量計費，不是完全免費，新帳號通常要先儲值
   （例如 5 美金），但每次辨識大概只花幾分錢等級

【怎麼貼到工具裡】
1. 打開工具，點「📷 辨識匯入」
2. 展開下方「▸ API 設定」
3. 貼上 API Key（存在自己這台電腦的瀏覽器裡，不會外傳、不會存到網站主機）
4. 換一台電腦、換一個瀏覽器要重貼一次；每個學員如果也要用這功能，
   也得各自申請、各自貼自己的 Key


----------------------------------------
常見問題
----------------------------------------

Q：git push 卡住沒反應？
A：通常是在等瀏覽器授權，切換到瀏覽器視窗，
   點「Authorize git-ecosystem」，回到 PowerShell 就會繼續。

Q：git status 出現 "not a git repository"？
A：代表這個資料夾遺失了 .git 隱藏資料夾（可能複製時漏了），
   需要重新設定，請找 Claude 協助處理。

Q：忘記資料夾搬去哪裡了？
A：打開檔案總管，點左側「本機磁碟(D:)」，
   在右上角搜尋框輸入「life-events」即可找到目前位置。

Q：辨識匯入按了說「請先在 API 設定填入 API Key」？
A：見上方「辨識匯入功能：需要 API Key」章節，先申請並貼上金鑰。
